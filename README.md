# cookiesGimnasio
Página web de un gimnasio con formulario de reserva de clases. Incluye validación de datos (nombre, tipo de clase y fecha), gestión de eventos, almacenamiento en cookies y funciones para cargar/eliminar reservas. Diseño responsivo con CSS y animaciones.

Gimnasio Plus Fitness - Aplicación Web de Gestión de Reservas

Esta es una aplicación web para gestionar reservas de clases en un gimnasio. Permite a los usuarios introducir sus datos, validar la información, guardar la reserva en cookies del navegador y gestionar sus citas almacenadas. Está desarrollada con HTML, CSS y JavaScript, siguiendo principios de usabilidad, validación de formularios y manipulación del DOM.

CARACTERÍSTICAS PRINCIPALES

Formulario interactivo con campos para nombre, tipo de clase y fecha.

Validación en tiempo real mediante expresiones regulares y comprobaciones lógicas.

Almacenamiento persistente de reservas utilizando cookies del navegador.

Funciones para cargar y borrar reservas guardadas.

Interfaz responsiva y moderna con animaciones CSS y diseño adaptable.

Manejo de eventos para envío de formularios y gestión de interacciones del usuario.

TECNOLOGÍAS UTILIZADAS

HTML5 para la estructura del contenido.

CSS3 para estilos, gradientes, animaciones y diseño responsivo.

JavaScript (ES6) para validaciones, manejo de cookies y lógica de la aplicación.

Expresiones regulares para validación de formatos de datos.

Uso de Flexbox y media queries para layout adaptable.

ESTRUCTURA DEL PROYECTO

El proyecto consta de tres archivos principales:

gimnasio.html: Contiene la estructura HTML de la aplicación.

estilos.css: Define todos los estilos visuales y animaciones.

script.js: Implementa la lógica de validación, cookies y gestión de eventos.

FUNCIONALIDADES IMPLEMENTADAS

Validación de nombre (solo letras y espacios).

Validación de tipo de clase (entre opciones predefinidas).

Validación de fecha (formato DD/MM/AAAA y fecha futura).

Guardado de reserva en cookie con caducidad.

Recuperación de datos desde cookie al recargar la página.

Borrado seguro de la reserva almacenada.

Mensajes de alerta para feedback al usuario.

Diseño optimizado para dispositivos móviles y escritorio.
